# 致学生

如果你是一名正在阅读本书的学生，谢谢你！能够为你探索操作系统知识的旅程提供一些帮助性的材料，我们深感荣幸。我们两人都会带着怀念回想起自己本科时期读过的一些教材（例如 Hennessy 和 Patterson 的 [HP90]——那本关于计算机体系结构的经典著作），也希望这本书将来能成为你记忆中这样一段美好的学习经历。

你可能已经注意到，这本书是免费并且可以在线获取的。[^2] 其中一个最主要的原因是：教材通常太贵了。我们希望，这本书能成为新一波免费教学材料中的先行者，为所有追求教育的人提供帮助——无论他们来自世界的哪个角落，也无论他们愿意为一本书花多少钱。即使做不到那么远大，至少这也是一本免费的书；而有一本免费的书，总比一本都没有要好。

我们也希望，在可能的情况下，把书中许多内容的原始来源指给你看：那些在多年间塑造了操作系统这一领域的重要论文，以及那些做出这些贡献的人。思想并不会凭空出现；它们来自聪明而勤奋的人们（其中也包括许多图灵奖得主）。[^3] 因此，只要有机会，我们就应当努力去致敬这些思想，以及提出这些思想的人。通过这样做，我们也许能更好地理解这个领域曾经发生过的那些变革，而不是把教材写得仿佛这些想法从来就是天经地义地存在着一样 [K62]。此外，我们也希望这些引用能鼓励你自己进一步深挖；阅读本领域那些著名的经典论文，毫无疑问是最好的学习方式之一。

[^2]: 这里稍微岔开说明一下：我们在这里所说的“免费（free）”，并不意味着“开源（open source）”，也不意味着这本书没有通常意义上的版权保护——它当然是受版权保护的！这里“免费”的意思只是：你可以下载这些章节，并用它们来学习操作系统。为什么不把它做成一本像 Linux 内核那样的开源图书呢？这是因为我们认为，一本书最好始终保持统一的叙述声音，而我们也一直在努力做到这一点。读这本书时，它应该有点像是在和一个正在向你解释问题的人对话。这就是我们采取这种方式的原因。

[^3]: 图灵奖是计算机科学领域的最高奖项；它大概就像计算机科学界的诺贝尔奖——只不过，你多半没听说过它。
